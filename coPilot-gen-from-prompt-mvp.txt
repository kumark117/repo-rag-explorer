FUNCTIONAL REQUIREMENTS
============================================================
1) Ingestion
- POST /api/ingest
- Body: { repoUrl: string }
- Download GitHub repo zip
- Extract and scan files recursively
- Include only: .ts .tsx .js .jsx .py .md
- Ignore: node_modules dist build
- Chunk files (500-800 chars)
- Embed chunks
- Store in in-memory vector store
- Return stats + file list

2) Query
- POST /api/query
- Body: { question: string }
- Embed question
- searchSimilar(topK=5)
- Send retrieved chunks + question to gpt-4o-mini
- Return answer + sources

3) RAG Prompt
Use this template:

Explain clearly and mention file names when possible.

  - Middle: Code Viewer panel

- ChatPanel sends POST /api/query with { question }
- Display AI response and simple loading/error state
