------------------------------------------------------------
OBJECTIVE
------------------------------------------------------------
Build an AI developer tool that analyzes GitHub repositories with Retrieval Augmented Generation (RAG):

1) Accepts a GitHub repository URL
2) Downloads the repository
3) Reads source files
4) Splits files into chunks
5) Generates embeddings
6) Stores embeddings in a vector store
7) Lets users ask questions about the codebase
8) Uses RAG to generate answers grounded in code
